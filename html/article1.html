Le but de notre projet ?

Nolan, Yannis 25/04/2020

Le but tout d'abbord de notre projet était de commentaire plusieurs codes que notre professeur nous a donnés en python, le but de ses codes était de pouvoir télécharger plusieurs images de différents chapitres dans un dossier donner. Ce travail de commentaire de texte nous a pris beaucoup de temps à mon camarade et moi-même mais nous nous sommes quand même parvenu. Après ce commentaire nos professeurs nous ont alors demandé de créer notre propre code avec des bouts de codes déjà existant dans leurs codes. Nous avons pensé que c'était alors facile mais nous nous sommes trompé c'était un travail assez dur qui nous a demandé beaucoup d'investissement

Bout de Code "Donwload"

Voici une partie du premier code commenter de notre professeur qui était le code principal de tous les autres On peut y voir différentes fonctions, comme dosser qui permet de choisir le fichier où vont aller les images mais aussi la fonction Dowload qui permet de télécharger les images
