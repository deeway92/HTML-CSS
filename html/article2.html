Comme je les dis précédemment dans l'article 1 nous avons réutilisé des bouts de code des codes python de notre professeur, que nous avons rassemblées, pour pouvoir télécharger d'autres images sur un autre site différents du sien. Avant que notre code marche nous avons eu bien sûr l'aide de M. Gibaud Et Gorce durant les heures de cours mais aussi particulièrement Jeras et Yassine. Cela nous a pris beaucoup de temps pour Yannis et Moi !

Voici notre code python composé de 4 fonctions

Notre code est alors composé de 4 fonctions =(Dossier, Naviguâtes, RecupListeLiens, Next, Class Download), la fonction dossier est ici pour choisir le fichier où les images vont être mises Naviguâtes permet d'aller sur un navigateur et entrer l'URL donner, RecupListeLiens permet de récupérer toutes les images dans une liste et de faire une recherche dans le fichier code du site en question, Next permet de changer de page. Pour que notre code fonctionne il faut alors dans un premier temps lancer le programme "Download", puis notre code, une fois ceci fait il faut entrer dans les shells (ex = Site) donc ceci puis Ex-site("URL du site) ici on définit le lien du site puis "Fichier'" et ici on définit où vont aller les images dans cet exemple les images vont aller dans le fichier "fichier". Une fois ceci fait il nous reste juste à taper en reprenant l'exemple ci-dessus Nolan.InitSoup(), et si tous ce passe bien les images vont allez se mettre dans le fichier !
